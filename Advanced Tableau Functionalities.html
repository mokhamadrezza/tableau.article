Advanced Analytics Scenarios dan Resources

Advanced Analytics pada Tableau

Advanced Analytics adalah pemeriksaan data atau konten secara otonom atau semi-otonom menggunakan teknik atau alat yang canggih (Faraz 2022). Teknik ini bertujuan untuk mengekstraksi informasi yang berguna untuk memahami tren dan perilaku yang direpresentasikan oleh data tersebut.

Notes

Teknik-teknik ini menggunakan Machine Learning dan Data Science seperti Predictive Analytics, Recommendation Systems, dan Statistical Forecasting untuk mengidentifikasi wawasan yang lebih dalam dari data.

Fitur Advanced Analytics dalam Tableau

Dalam Tableau, advanced analytics mencakup fitur-fitur berikut:

  • Segmentation dan Cohort Analysis
  • Scenario and What-If Analysis
  • Sophisticated Calculations and Statistical Functions
  • Time-Series and Predictive Analytics
  • R and Python Integration

Segmentation dan Cohort Analytics

Cohort Analysis adalah analisis unit tentang perilaku user, subscriber, atau konsumen, yang dilakukan secara mendalam untuk mendapatkan data terkait tingkat interaksi pelanggan maupun calon pelanggan.

Notes

Cohort Analysis menggunakan fitur demografis umum yang difokuskan dalam kurun waktu tertentu dan bisa menjadi strategi untuk mengembangkan bisnis.

Beberapa tipe Cohort Analysis:

  1. Behavioral Cohort: Mengkategorikan kelompok konsumen berdasarkan perilaku mereka.
  2. Acquisition Cohort: Menganalisis data berdasarkan waktu akuisisi konsumen.
  3. Specific Customer Behavior: Menganalisis perilaku konsumen berdasarkan kelompok yang berbeda.

Untuk menggunakan cohort analysis:

  1. Pilih Analysis > Create Calculated Field
  2. Buat acquisition cohort dengan memasukkan sintaks berikut:

    3. DATETRUNC('month', {FIXED (CUSTOMER ID): MIN([Order Date])})

  3. Klik OK dan cohort siap digunakan.

Scenario and What-If Analysis

Analisis What-If adalah simulasi data intensif untuk memeriksa perilaku sistem yang kompleks. Di Tableau, ini dilakukan dengan menggunakan parameter dan kalkulasi. Contoh: Membuat parameter untuk tingkat inflasi dan menggunakan parameter ini dalam kalkulasi perkiraan penjualan.

Jenis dari What-If Analysis:

  1. IF: Mengembalikan hasil jika kondisi terpenuhi.

    2. IF [Sales] > 1000 THEN 'High Sales' END

  2. IF − ELSE: Mengembalikan nilai yang telah ditetapkan jika kondisi tidak terpenuhi.

    3. IF [Sales] > 1000 THEN 'High Sales' ELSE 'Low Sales' END

  3. ELSE IF: Mengevaluasi kondisi tambahan jika kondisi dalam pernyataan if sebelumnya bernilai false.

    4. IF [Sales] > 1000 THEN 'High Sales' ELSEIF [Sales] > 500 THEN 'Medium Sales' ELSE 'Low Sales' END

What-If-Analysis dapat digunakan dengan cara berikut:

  1. Pilih Analysis > Create Calculated Field
  2. Buat What-If-Analysis dengan membuat pernyataan ELSEIF menggunakan sintaks seperti di bawah ini:

    3. IF [Sales] > 1000 THEN 'High Sales' ELSEIF [Sales] >= 500 THEN 'Medium Sales' ELSE 'Low Sales' END

  3. Klik OK dan siap digunakan.

Sophisticated Calculations dan Statistical Functions

Tableau menyediakan berbagai fitur untuk melakukan perhitungan canggih dan fungsi statistik:

  1. Calculated Fields: Menghasilkan data baru dari data yang sudah ada.
  2. Perhitungan Tabel: Menggunakan fungsi agregasi seperti jumlah, rata-rata, persentil, dll.
  3. Fungsi LOD (Level of Detail): Melakukan perhitungan pada level detail yang berbeda.
  4. Perhitungan Prediktif: Menggunakan MODEL_QUANTILE dan MODEL_PERCENTILE.
  5. Fungsi Statistik: Menyediakan fungsi seperti COVAR, CORR, MEDIAN, PERCENTILE, dll.
  6. Perhitungan Logika dan String: Menggunakan pernyataan logika seperti IF, ELSE IF, dan ELSE, atau fungsi string seperti CONTAINS, STARTSWITH, dan FIND.

Sophisticated Calculation dan Statistical Function dapat ditemukan pada saat pengguna:

  1. Memilih Analysis > Create Calculated Field
  2. Klik tombol panah kanan pada calculated field, maka sophisticated calculation dan statistical function akan muncul.

Integrasi dengan R dan Python

Tableau menyediakan integrasi dengan bahasa pemrograman R dan Python untuk analisis data yang lebih lanjut. Perbedaan antara R dan Python:

  1. Bahasa Pemrograman R:
    1. Digunakan untuk analisis statistik.
    2. Banyak digunakan oleh akademisi dan peneliti.
    3. Memberikan fleksibilitas menggunakan library yang tersedia.
  2. Bahasa Pemrograman Python:
    1. Umumnya digunakan untuk keperluan data science.
    2. Banyak digunakan oleh programming atau developer.
    3. Memberikan fleksibilitas untuk membangun model baru dari awal.

Untuk melakukan integrasi ini, pengguna harus mengikuti langkah-langkah:

  1. Pilih Help > Settings and Performance > Manage Analytics Extension Connection.
  2. Kemudian akan muncul window yang berisi beberapa connection termasuk R dan Python.

Membuat Kalkulasi pada Tableau

Menghubungkan Data ke Tableau

Dataset Superstore adalah salah satu open dataset yang tersedia pada platform Tableau yang memuat informasi tentang penjualan dan keuntungan Superstore. Untuk menghubungkan dataset ke Tableau:

  1. Buka Tableau: Pilih "Microsoft Excel" pada panel "Connect".
  2. Pilih Lembar Kerja: Pilih file Excel yang diinginkan.
  3. Impor Data: Pilih lembar kerja "Orders" atau bisa drag-and-drop lembar kerja tersebut ke bagian bawah label "Global Superstore".

Visualisasi Kalkulasi Sederhana

Untuk membuat kalkulasi sederhana di Tableau:

  1. Buat Calculated Field:
    1. Pilih Analisis > Create Calculated Field.
    2. Beri nama calculated field, misalnya "Profit Ratio".
    3. Masukkan rumus: SUM([Profit])/SUM([Sales]).
    4. Klik OK.
  2. Tambahkan Calculated Field ke Measures: Kolom baru akan ditambahkan ke panel Data.

Kalkulasi Tabel pada Tableau

  1. Pilih Data: Dari data panel, pilih “Sales” dan “Order Date”.
  2. Buat Tabel Kalkulasi: Pilih tabel pada panel "Show Me".
  3. Ubah Format: Drag-and-drop “Order Date” ke panel “Rows” dan ubah dari “Year” ke “Month” atau “Quarter”.

Level Of Detail (LOD) Expressions pada Tableau

LOD expressions memungkinkan pengguna menghitung nilai pada tingkat detail yang berbeda:

  1. Pilih Data:
    1. Buat worksheet baru.
    2. Drag and drop “Region” ke panel “Column” dan “Sales” ke “Rows”.
  2. Buat LOD Calculation:
    1. Pilih Analysis > Create Calculated Field.
    2. Beri nama kalkulasi, misalnya "Sales Per Customer".
    3. Masukkan ekspresi LOD: { INCLUDE [Customer Name] : SUM([Sales]) }.
    4. Klik OK.
  3. Tambahkan LOD Calculation ke Visualisasi:
    1. Drag-and-drop “Sales Per Customer” ke panel “Rows”.
    2. Pilih “Measure (Sum)” > “Average”.
