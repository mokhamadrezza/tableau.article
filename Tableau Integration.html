Integrasi dengan Python

Tableau Integration Feature

Tableau Integration Feature adalah fitur yang memungkinkan Tableau untuk terhubung dan terintegrasi dengan aplikasi, bahasa pemrograman, database, dan file lainnya (Tableau n.d). Fitur ini menyediakan konektivitas, integrasi, dan ekstensibilitas yang fleksibel bagi pengguna Tableau. Fitur integrasi Tableau mengacu pada kemampuan perangkat lunak Tableau untuk terhubung dan berinteraksi pada:

  • Local Database
  • Remote Database
  • Spreadsheet
  • Text File
  • Aplikasi Cloud
  • Bahasa pemrograman (untuk komputasi dan analisis)
  • Kueri SQL

Mengenal Python

Python merupakan salah satu bahasa pemrograman yang banyak digunakan oleh perusahaan besar maupun para developer untuk mengembangkan berbagai macam aplikasi berbasis desktop, web, dan mobile (Romzi & Kurniawan 2020). Python diciptakan oleh Guido van Rossum di Belanda pada tahun 1990 dan namanya diambil dari acara televisi kesukaan Guido, Monty Python's Flying Circus.

Python adalah bahasa pemrograman tingkat tinggi yang dirancang untuk tujuan umum seperti dalam pengembangan web, automasi, software testing, analisis data, machine learning, dan pengembangan game. Bahasa pemrograman ini memiliki filosofi desain yang menekankan keterbacaan kode dengan penggunaan indentasi yang signifikan.

Tableau Integration with Python

Integrasi Python pada Tableau memungkinkan pengguna Tableau untuk mengeksekusi kode Python dan fungsi yang disimpan di dalamnya. Hal ini memungkinkan pengguna untuk memanfaatkan kekuatan Python dalam analisis data dan visualisasi data di dalam Tableau. Integrasi Python pada Tableau dapat dilakukan dengan menggunakan TabPy Server.

TabPy Server adalah server Python yang dapat diakses dari Tableau. Pengguna dapat mengeksekusi kode Python dan fungsi yang disimpan di dalamnya melalui TabPy Server (Bima Putra, 2020).

Notes

Dalam integrasi Python pada Tableau, pengguna harus menginstal Python dan memastikan bahwa Python dapat diakses dari Tableau. Selain itu, pengguna juga harus memastikan bahwa kode Python dan fungsi yang disimpan di dalamnya dapat diakses dari Tableau.

Langkah-langkah Integrasi Python pada Tableau (Windows):

  1. Instal Python:

    2. Jika belum terinstal, buka laman https://www.python.org

  2. Instalasi Python:
    1. Buka file instalasi Python yang telah terunduh, lalu checklist “Add Python [version] to PATH”.
    2. Pilih "Install Now" dan tunggu hingga proses instalasi selesai.
  3. Verifikasi Instalasi Python:
    1. Tekan Windows+R pada keyboard.
    2. Ketik “cmd” dan pilih OK untuk membuka “Command Prompt”.
    3. Pada “Command Prompt” ketik “python --version” dan tekan enter untuk melihat versi Python yang terinstal.
  4. Install TabPy Server:
    1. Pada Command Prompt, ketik “pip install tabpy-server” dan tekan enter.
    2. Tunggu proses instalasi modul TabPy Server selesai.
  5. Konfigurasi TabPy Server:
    1. Ganti direktori di Command Prompt dengan mengetik

      “cd C:\Users\[nama user]\AppData\Local\Programs\Python\Python[versi terinstal]\Lib\site-packages\tabpy_server”

    2. Buka file “startup.bat” untuk mengkonfigurasikan Python pada Tableau untuk pertama kalinya.
    3. Ketikkan “y” untuk memulai TabPy.
  6. Integrasi dengan Tableau:
    1. Buka Tableau dan masukkan data Global Superstore pada sheet Order.
    2. Pada menu Help > Setting and Performance > pilih TabPy.
    3. Pada window Manage Analytics Extension Connection, masukkan “localhost” pada Hostname dan 9004 pada port, lalu pilih Save.
    4. Klik “Test Connection” untuk memastikan koneksi berhasil.
  7. Verifikasi Integrasi:
    1. Pilih Analysis > Create Calculated Field.
    2. Masukkan nama “Profitable” lalu masukkan sintaks berikut:

      3. SCRIPT_BOOL("

      Mylist = []

      for x in _arg1:

        Mylist.append(x > 0)

      return Mylist"

      , SUM([Profit]))

      Sehingga akan terbentuk satu variabel baru yaitu Profitable.

Integrasi dengan R

Mengenal R (Programming Language)

R adalah software program yang berada di bawah GPL (General Public License), yang artinya software ini dapat digunakan secara bebas atau tidak perlu membayar lisensi untuk menggunakannya (Fauzan, Mahara & Sulistyo 2020). Program R adalah sistem yang digunakan untuk analisis statistika dan relatif lengkap. R memiliki sintaks yang mirip dengan bahasa pemrograman S dan menyediakan lingkungan perangkat lunak yang lengkap untuk analisis data statistik.

R mendukung analisis dan visualisasi data yang interaktif, termasuk grafik 2D dan 3D, dan animasi. Sebagai contoh dalam analisis perilaku pasar, forecasting, dan topic modelling. R juga dapat diintegrasikan dengan aplikasi lain, seperti Tableau, untuk memperluas kemampuan analisis data dan visualisasi data.

Tableau Integration with R

Integrasi R pada Tableau memungkinkan pengguna Tableau untuk mengeksekusi kode R dan fungsi yang disimpan di dalamnya. Hal ini memungkinkan pengguna untuk memanfaatkan kekuatan R dalam analisis data dan visualisasi data di dalam Tableau. Integrasi R dengan Tableau dapat dilakukan dengan membutuhkan paket eksternal yaitu Rserve.

Rserve adalah paket R yang memungkinkan pengguna untuk menghubungkan R dengan aplikasi lain, termasuk Tableau. Rserve memungkinkan pengguna untuk mengeksekusi kode R dan fungsi yang disimpan di dalamnya melalui Tableau (Samuel 2022).

Notes

Dalam integrasi R pada Tableau, pengguna harus menginstal dan memastikan bahwa R dapat diakses dari Tableau. Selain itu, pengguna juga harus memastikan bahwa kode R dan fungsi yang disimpan di dalamnya dapat diakses dari Tableau.

Langkah − Langkah Integrasi R pada Tableau

  1. Install R:

    2. Jika belum terinstal, buka laman https://cran.r-project.org/ dan download versi terbaru.

  2. Instalasi R:
    1. Buka file instalasi yang telah terunduh dan pilih bahasa yang akan digunakan untuk proses instalasi.
    2. Lanjutkan proses instalasi dan tunggu hingga selesai.
  3. Install Rserve:
    1. Buka program R versi terbaru.
    2. Ketik “install.packages(“Rserve”)” pada command line dan tekan enter.
    3. Pilih CRAN mirrors “0-Cloud [https]” untuk memilih default server CRAN tempat mendownload paket.
  4. Panggil dan Jalankan Rserve:
    1. Ketik “library(Rserve)” lalu tekan enter.
    2. Jalankan Rserve dengan mengetik “Rserve()” dan tekan enter.
  5. Integrasi dengan Tableau:
    1. Buka Tableau dan masukkan data Global Superstore pada sheet Order.
    2. Pada menu Help > Setting and Performance > pilih Rserve.
    3. Pada window Manage Analytics Extension Connection, masukkan “localhost” pada Hostname dan 6311 pada port, lalu pilih Save.
    4. Klik “Test Connection” untuk memastikan koneksi berhasil.
  6. Verifikasi Integrasi:
    1. Pilih Analysis > Create Calculated Field.
    2. Masukkan nama “Profitable” lalu masukkan sintaks berikut:

      3. SCRIPT_BOOL("

      Mylist ← list()

      for (x in _arg1) {

        Mylist ← append(Mylist, list(x > 0)){

      return(Mylist)"

      , SUM([Profit]))

      Sehingga akan terbentuk satu variabel baru yaitu Profitable.
