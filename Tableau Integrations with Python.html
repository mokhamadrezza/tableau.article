Python pada Tableau

Perbedaan Tableau dan Python

Tableau dan Python adalah dua hal yang berbeda dalam konteks pemrograman dan analisis data. Tableau merupakan tools yang dapat digunakan untuk membuat visualisasi data ke dalam bentuk dashboard, sehingga data dapat menjadi lebih interaktif, dan mudah untuk dianalisis (Afikah, Afandi & Hasan 2022). Sedangkan Python adalah bahasa pemrograman yang digunakan untuk analisis data, pemrosesan data, dan pengembangan aplikasi. Berikut adalah beberapa perbedaan antara Tableau dan Python:

Komponen Tableau Python
Penggunaan Dapat terhubung ke berbagai jenis database dan dapat memuat data dari berbagai jenis file seperti teks, JSON, CSV, xlsx, dll Mengelola data yang sangat besar bahkan jika melalui streaming data sekalipun
Penanganan Data Cepat dan mudah dipelajari. Selain itu, kemudahannya dalam pengoperasian hampir tidak memerlukan pengetahuan pemrograman yang mendalam Memakan waktu dan cukup sulit dipelajari dikarenakan memerlukan pengetahuan pemrograman
Visualisasi Mudah digunakan oleh user dalam melakukan visualisasi data Dapat menghasilkan visualisasi, tetapi prosesnya sangat memakan waktu dan rumit
Bahasa Pemrograman Menggunakan bahasa pemrograman yang disebut Tableau Calculation Language (TCL) Menggunakan bahasa pemrograman Python
Kompleksitas Lebih mudah digunakan dan dipelajari Memerlukan pemahaman yang lebih mendalam tentang bahasa pemrograman dan konsep pemrograman

Penggunaan Python dan Tableau

Terdapat beberapa cara untuk mengintegrasikan Python pada Tableau, seperti menggunakan TabPy. Integrasi Python pada Tableau memungkinkan beberapa hal yaitu:

  1. Pengguna Tableau dapat mengeksekusi kode Python dan fungsi yang disimpan di dalamnya, sehingga pengguna dapat memanfaatkan kekuatan Python dalam analisis data dan visualisasi data di dalam Tableau.
  2. Pengguna dapat memanfaatkan kekuatan Tableau dalam memvisualisasikan data dan membuat dashboard.

Dalam memilih antara Tableau dan Python, pengguna harus mempertimbangkan tujuan penggunaan, kompleksitas, kustomisasi, dan integrasi dengan aplikasi lain. Python dan Tableau dapat digunakan bersama-sama untuk meningkatkan analisis data dan kemampuan visualisasi data. Berikut adalah cara terhubung ke sumber data, memanipulasi data, dan membuat visualisasi khusus menggunakan Python dan Tableau:

  1. Terhubung ke Sumber Data

    2. Tableau dapat terhubung dengan berbagai sumber data seperti file Excel, CSV, dan database SQL. Untuk terhubung ke sumber data Python, kita dapat menggunakan library seperti Pandas dan SQLAlchemy. Pandas dapat membaca file CSV, Excel, SQL, dan berbagai format data lainnya. SQLAlchemy dapat digunakan untuk terhubung ke database SQL dan melakukan query.

  2. Manipulasi Data

    3. Setelah terhubung ke sumber data, pengguna dapat menggunakan Pandas untuk memanipulasi data sebelum dimasukkan ke dalam Tableau. Contohnya, melakukan transformasi data seperti penghapusan kolom atau baris yang tidak diperlukan, penggabungan data, atau pembuatan kolom baru dari data yang sudah ada.

  3. Visualisasi Khusus

    4. Tableau memiliki berbagai pilihan visualisasi, namun dengan menggunakan Python memungkinkan untuk membuat visualisasi yang lebih khusus dan lebih kompleks. Pengguna dapat menggunakan library seperti Matplotlib, Seaborn, dan Plotly untuk membuat visualisasi yang lebih kustom. Data dari Python dapat diintegrasikan ke dalam visualisasi Tableau dengan menggunakan konsep yang disebut Tableau External Service.

    Untuk menggunakan Tableau External Service, pengguna perlu membuat script Python dan menghubungkannya ke Tableau. Setelah itu, membuat tindakan khusus pada visualisasi Tableau untuk memanggil script Python dan menampilkan visualisasi yang dihasilkan dalam jendela pop-up atau dalam visualisasi Tableau.

Visualisasi dengan Tabpy

Pengenalan Tabpy

TabPy (Server Tableau Python) adalah implementasi Analytics Extension yang memperluas kemampuan Tableau dengan memungkinkan pengguna untuk mengeksekusi skrip Python dan menyimpan fungsi melalui perhitungan tabel Tableau (Fridriksson 2019). Framework TabPy memungkinkan Tableau untuk mengeksekusi kode Python dari jarak jauh.

TabPy memiliki dua komponen, yaitu:

  1. Serangkaian REST API yang memungkinkan eksekusi kode jarak jauh dalam proses server dan diekspos sebagai titik akhir REST, untuk dipanggil nanti.
  2. Tools library, berdasarkan fungsi Python yang memungkinkan penerapan dengan tingkat fleksibilitas yang tinggi.

Seiring dengan dirilisnya API (Application Programming Interface), Layanan Eksternal Python, R, dan MATLAB Tableau yang sudah ada dikenal sebagai "Analytics Extensions". API Analytics Extensions didasarkan pada API Layanan Eksternal TabPy asli, dan TabPy dapat dianggap sebagai implementasi API referensi Tableau. Terdapat dua cara yang dapat digunakan untuk melakukan perhitungan Python:

  1. Tulis kode secara langsung, kemudian kode akan dieksekusi di server TabPy.
  2. Terapkan fungsi ke TabPy server yang dapat dijangkau sebagai REST API endpoint melalui otentikasi dasar dan SSL.

Langkah-Langkah Menggunakan Tabpy

Dengan menggunakan TabPy, pengguna dapat melakukan berbagai hal salah satunya yaitu untuk mencari nilai korelasi. Sebagai contoh, untuk memvisualisasikan korelasi antara Price dengan Number of Reviews dan Minimum Nights pada data AirBnB. Dataset AirBNB adalah data berisi nama pemilik hotel, harga, tipe ruangan dan lain-lain pada penginapan AirBnB New York 2019. Berikut adalah langkah-langkahnya:

  1. Unduh dataset AirBNB yang akan digunakan. Dataset tersedia pada: AirBnB Dataset .
  2. Import dataset yang telah diunduh, dari 16 kolom, akan digunakan variabel berikut dalam kumpulan data ini untuk melakukan mencari korelasi:
    1. price
    2. minimum_nights
    3. number_of_reviews
  3. Jalankan TabPy dengan mengetikkan “tabpy” pada Command Prompt lalu enter.
  4. Buka Tableau dan pilih menu Help > Setting and Performance > TabPy.
  5. Pada window Manage Analytics Extension Connection, masukkan “localhost” pada Hostname dan 9004 pada Port dan pilih Save
  6. Untuk memastikan apakah koneksi berhasil dapat mengklik “Test Connection” dan akan muncul pop-up seperti gambar di bawah ini.
  7. Menampilkan Korelasi dengan Tabpy
    1. Untuk menampilkan korelasi antara jumlah review dengan harga, pada menu pilih Analysis > Create Calculated Field.

      2. Masukkan nama “Correlation” lalu masukkan sintaks berikut:

      SCRIPT_REAL("import numpy as np return"

      np.corrcoef(_arg1,_arg2)[0,1]", SUM([Price]),

      SUM([Number Of Reviews]))

    2. Sebelum menekan OK, pilih pada “Default Table Calculation” dan ganti opsi dari Automatic ke Host Id, lalu klik OK.
    3. Drag-and-drop “Price” dan “Room Type” pada panel “Columns”.
    4. Drag-and-drop “Number of Review” pada panel “Rows”.
    5. Drag-and-drop “Host Id” dan “Correlation” pada panel “Marks Card”.
    6. Pilih ikon pada “Correlation” yang terletak di “Mark Card” dan ganti menjadi “Color”.
    7. Pilih pada “Correlation” yang terletak di “Mark Card”, lalu pilih “Discrete” untuk mengganti step warna pada legenda.
  8. Menampilkan Korelasi Minimum Nights dan Price
    1. Untuk menampilkan korelasi antara jumlah minimal malam menginap dengan harga, pada menu pilih Analysis > Create Calculated Field.
    2. Masukkan nama “Correlation Night-Price” lalu masukkan sintaks berikut:

      3. SCRIPT_REAL("import numpy as np

      return np.corrcoef(_arg1,_arg2)[0,1]",

      SUM([Price]),SUM([Minimum Nights]))

    3. Sebelum menekan OK, pilih pada “Default Table Calculation” dan ganti opsi dari Automatic ke Host Id, lalu klik OK.
    4. Drag and drop “Price” dan “Room Type” pada panel “Columns”.
    5. Drag and drop “Minimum Nights” pada panel “Rows”.
    6. Drag and drop “Host Id” dan “Correlation Night-Price” pada panel “Marks Card”.
    7. Pilih ikon pada “Correlation Night-Price” yang terletak di “Mark Card” dan ganti menjadi “Color”.
    8. Pilih pada “Correlation Night-Price” yang terletak di “Mark Card”, lalu pilih “Discrete” untuk mengganti step warna pada legenda.
