R pada Tableau

Perbandingan Tableau dengan R

Tableau merupakan tools yang dapat digunakan untuk membuat visualisasi data ke dalam bentuk dashboard, sehingga data dapat menjadi lebih interaktif, dan mudah untuk dianalisis (Afikah, Afandi & Hasan 2022). Sedangkan program R adalah bahasa pemrograman yang dapat digunakan secara bebas atau tidak perlu membayar lisensi (Fauzan, Mahara & Sulistyo 2020).

Tableau dan R adalah dua hal yang berbeda dalam konteks pemrograman, visualisasi, dan analisis data. Berikut adalah beberapa perbedaan antara R dan Tableau:

Komponen Tableau R
Kegunaan Digunakan untuk membangun visualisasi data yang interaktif Digunakan untuk berbagai hal seperti web extraction, machine learning, statistik, visualisasi data, web-apps, dan lain-lain
Kurva Belajar (Learning Curve) Cepat dan mudah dipelajari. Selain itu, kemudahannya dalam pengoperasian hampir tidak memerlukan pengetahuan pemrograman yang mendalam Memakan waktu dan cukup sulit dipelajari dikarenakan memerlukan pengetahuan pemrograman
Pengoperasian Visualisasi Menyediakan visualisasi yang interaktif bagi pengguna maupun audiens secara langsung Sepenuhnya tidak berbayar dikarenakan R merupakan open-source pemrograman
Kontribusi Pihak Ketiga Merupakan program berbasis Graphical User Interface (GUI), sehingga membatasi kontribusi pihak ketiga (orang lain) dalam mengembangkan program mereka Merupakan program berbasis Command Line Interface (CLI), sehingga pihak eksternal (orang lain) dapat berkontribusi secara bebas terhadap pengembangan program R

Integrasi R dan Tableau

Meskipun memiliki perbedaan, terdapat beberapa cara untuk mengintegrasikan R dan Tableau untuk penggunaan secara bersama-sama, seperti dengan menggunakan bantuan paket Rserve.

Rserve adalah paket R yang memungkinkan aplikasi lain untuk berkomunikasi dengan R menggunakan TCP/IP atau soket lokal (Rforge, n.d).

Berikut adalah beberapa cara untuk mengintegrasikan Rserve dengan Tableau:

  1. Menghubungkan Tableau Prep ke Rserve: Pengguna dapat menghubungkan Tableau Prep ke Rserve untuk melakukan analisis data dan memvisualisasikan data. Pengguna dapat mengikuti tutorial pada dokumentasi resmi dari Tableau untuk menghubungkan Tableau Prep ke Rserve.
  2. Mengkonfigurasi Tableau Server untuk R dan Rserve: Pengguna dapat mengkonfigurasi Tableau Server untuk terhubung ke R dan Rserve melalui TSM command line/shell. Setelah itu, pengguna dapat membuat dashboard yang mengandung fungsi R dan menampilkannya dalam visualisasi Tableau.
  3. Menggunakan R dengan Tableau Desktop: Pengguna dapat menghubungkan R dengan Tableau Desktop melalui calculated fields.

Integration R with Tableau

Penggunaan Rserve

Integrasi Rserve dengan Tableau memungkinkan pengguna untuk memanfaatkan kekuatan R dalam analisis data dan visualisasi data di dalam Tableau. Untuk menggunakan R dalam Tableau dapat dilakukan dengan menggunakan paket Rserve, di mana integrasi ini memungkinkan pengguna untuk menggunakan dimensi dan ukuran data yang terhubung secara langsung dalam skrip R dan mengeksekusinya secara langsung. Setelah itu, pengguna dapat menggunakan hasil skrip dan menampilkannya dalam visualisasi Tableau.

Contoh Penggunaan R dengan Tableau

Sebagai contoh, untuk memvisualisasikan korelasi antara Price dengan Number of Reviews dan Minimum Nights pada data AirBnB. Dataset AirBNB adalah data berisi nama pemilik hotel, harga, tipe ruangan dan lain-lain pada penginapan AirBnB New York 2019. Berikut adalah langkah-langkahnya:

  1. Unduh dataset AirBNB yang akan digunakan. Dataset tersedia pada: AirBnB Dataset .
  2. Import dataset yang telah diunduh, dari 16 kolom, akan digunakan variabel berikut dalam kumpulan data ini untuk melakukan mencari korelasi:
    1. price
    2. minimum_nights
    3. number_of_reviews
  3. Buka program R dan panggil paket/modul Rserve dengan mengetikkan “library(Rserve)” pada R Console lalu enter.
  4. Jalankan Rserve dengan mengetikkan “Rserve()” pada R Console lalu enter.
  5. Buka Tableau dan pilih menu Help > Setting and Performance > Rserve.
  6. Pada window Manage Analytics Extension Connection, masukkan “localhost” pada Hostname dan 6311 pada Port dan pilih Save.
  7. Untuk memastikan apakah koneksi berhasil dapat mengklik “Test Connection” dan akan muncul pop-up seperti gambar di bawah ini.
  8. Menampilkan Korelasi
    1. Untuk menampilkan korelasi antara jumlah review dengan harga, pada menu pilih Analysis > Create Calculated Field.

      2. Masukkan nama “Correlation” lalu masukkan sintaks berikut:

      SCRIPT_REAL("

      cor(.arg1, .arg2)

      ", SUM([Price]), SUM([Number Of Reviews]))

    2. Sebelum menekan OK, pilih pada “Default Table Calculation” dan ganti opsi dari Automatic ke Host Id, lalu klik OK.
    3. Drag-and-drop “Price” dan “Room Type” pada panel “Columns”.
    4. Drag-and-drop “Number of Review” pada panel “Rows”.
    5. Drag-and-drop “Host Id” dan “Correlation” pada panel “Marks Card”.
    6. Pilih ikon pada “Correlation” yang terletak di “Mark Card” dan ganti menjadi “Color”.
    7. Pilih pada “Correlation” yang terletak di “Mark Card”, lalu pilih “Discrete” untuk mengganti step warna pada legenda.
  9. Menampilkan Korelasi Minimum Nights dan Price
    1. Untuk menampilkan korelasi antara jumlah minimal malam menginap dengan harga, pada menu pilih Analysis > Create Calculated Field.
    2. Masukkan nama “Correlation Night-Price” lalu masukkan sintaks berikut:

      3. SCRIPT_REAL("

        cor(.arg1, .arg2)

      ", SUM([Price]), SUM([Minimum Nights]))

    3. Sebelum menekan OK, pilih pada “Default Table Calculation” dan ganti opsi dari Automatic ke Host Id, lalu klik OK.
    4. Drag and drop “Price” dan “Room Type” pada panel “Columns”.
    5. Drag and drop “Minimum Nights” pada panel “Rows”.
    6. Drag and drop “Host Id” dan “Correlation Night-Price” pada panel “Marks Card”.
    7. Pilih ikon pada “Correlation Night-Price” yang terletak di “Mark Card” dan ganti menjadi “Color”.
    8. Pilih pada “Correlation Night-Price” yang terletak di “Mark Card”, lalu pilih “Discrete” untuk mengganti step warna pada legenda.
