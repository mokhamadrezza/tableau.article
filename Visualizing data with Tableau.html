Dataset dalam Tableau

Definisi Dataset

Dataset adalah kumpulan data yang disusun secara terstruktur dan terdiri dari satu atau lebih catatan/data. Dataset biasanya dipresentasikan dalam bentuk tabel, yaitu baris dan kolom, dan berisi lebih dari satu variabel yang menyangkut suatu topik tertentu. Dataset menjadi dasar bagi proses pengambilan keputusan yang berdasarkan data, yang memungkinkan pengguna untuk menggali wawasan baru, mengidentifikasi tren, dan menjawab pertanyaan bisnis melalui visualisasi yang mudah dimengerti. Dataset dalam Tableau dapat diperoleh dari berbagai sumber data seperti Excel, CSV, remote database, dan lain-lain. Dengan menggunakan dataset, pengguna Tableau dapat membuat dashboard yang menggabungkan beberapa visualisasi untuk memberikan pandangan menyeluruh tentang informasi yang terkandung dalam data.

Jenis-Jenis Dataset

Dalam Tableau, berbagai jenis dataset dapat digunakan untuk membuat visualisasi dan analisis yang kaya. Berikut adalah beberapa jenis dataset umum yang dapat digunakan dalam Tableau:

  1. Numerical Dataset

    2. Numerical dataset adalah jenis dataset yang terdiri dari data angka atau nilai numerik. Dataset ini terdiri dari kolom-kolom yang berisi angka-angka yang dapat diukur atau dihitung. Jenis dataset ini sangat umum dalam analisis data dan visualisasi karena angka-angka numerik memungkinkan untuk dilakukan berbagai jenis perhitungan, perbandingan, dan analisis statistik.

    Contoh dataset numerik:

    • Data Keuangan, seperti pendapatan, biaya, dan laba.
    • Data Waktu, seperti harga saham harian.
    • Data Ilmiah, seperti suhu, tekanan, atau kecepatan.
    • Data Statistik, seperti skor ujian siswa

    Dalam Tableau, dataset numerik dapat digunakan untuk membuat berbagai jenis visualisasi, seperti:

    1. Grafik Batang (Bar Chart): Membandingkan nilai numerik di antara kategori atau periode waktu yang berbeda.
    2. Diagram Garis (Line Chart): Menggambarkan perubahan nilai numerik seiring waktu atau variabel lain.
    3. Histogram: Memvisualisasikan distribusi dta numerik dengan mengelompokkan nilai-nilai dalam interval dan menunjukkan frekuensi kemunculan di setiap interval.
    4. Box Plot: Memberikan gambaran tentang distribusi, kuartil, dan nilai-nilai ekstrem dalam data numerik.
    5. Heatmap: Memvisualisasikan matriks data numerik dengan warna yang menggambarkan nilai intensitas.
    6. Scatter Plot: menunjukkan hubungan antara dua variabel numerik dan dapat membantu mengidentifikasi pola atau korelasi di antara mereka.

  2. Categorical Dataset

    3. Categorical dataset adalah jenis dataset yang terdiri dari data yang dikelompokkan ke dalam kategori atau label diskret. Data kategori ini mewakili atribut atau variabel yang tidak memiliki urutan alami, tetapi menggambarkan atribut atau kelompok yang berbeda. Dataset kategorikal sering digunakan untuk menganalisis data yang dapat dijelaskan dalam bentuk klasifikasi atau pengelompokan.

    Contoh dataset kategorikal:

    • Data jenis kelamin (misalnya, “Laki-laki” dan “Perempuan”).
    • Kategori produk.
    • Nama kota atau negara.
    • Tingkat pendidikan (seperti “SD”, “SMP”, “SMA”).
    • Status pernikahan (“Belum Menikah”, “Menikah”, “Cerai”).

    Dalam Tableau, dataset kategorikal dapat digunakan untuk membuat berbagai jenis visualisasi, seperti:

    1. Grafik batang (Bar Chart): Membandingkan jumlah atau proporsi dalam kategori yang berbeda.
    2. Diagram Lingkaran (Pie Chart): Menunjukkan proporsi setiap kategori dalam keseluruhan.
    3. Bar Chart Grouped/Stacked: Membantu membandingkan kategori-kategori di beberapa dimensi yang berbeda.
    4. Peta Choropleth: Jika data kategorikal terkait dengan lokasi geografis, memvisualisasikan distribusi kategori di berbagai wilayah.
    5. Word Cloud: Memvisualisasikan frekuensi kata-kata atau label dalam dataset kategorikal.

  3. Temporal Dataset

    4. Temporal dataset adalah jenis dataset yang berfokus pada data yang memiliki dimensi waktu atau tanggal. Data dalam dataset ini dicatat dan dikumpulkan pada titik-titik waktu tertentu, dan biasanya berurutan secara kronologis. Analisis data temporal memungkinkan pemahaman tentang bagaimana nilai-nilai atau peristiwa berubah seiring berjalannya waktu, serta pengidentifikasian tren, pola musiman, atau peristiwa khusus.

    Contoh dataset temporal:

    • Data harga saham harian.
    • Data cuaca harian.
    • Data penjualan bulanan.
    • Data penggunaan situs web per jam.
    • Data populasi tahunan.

    Dalam Tableau, dataset temporal memungkinkan untuk membuat berbagai jenis visualisasi dan analisis berbasis waktu, seperti:

    1. Line Chart: Memplot data berdasarkan dimensi waktu.
    2. Area Chart: Menunjukkan data seiring waktu dengan area di bawah garis yang diwarnai.
    3. Bar Chart dengan Waktu: Membandingkan nilai numerik dalam kategori waktu yang berbeda.
    4. Gantt Chart: Menggambarkan jadwal atau rangkaian peristiwa dalam bentuk grafik batang sepanjang waktu.

  4. Textual Dataset

    5. Textual dataset adalah jenis dataset yang berisi data dalam bentuk teks. Dataset ini terdiri dari kalimat, paragraf, atau dokumen yang dapat berisi informasi, deskripsi, ulasan, komentar, atau konten teks lainnya. Analisis teks bertujuan untuk menggali wawasan dari data teks ini, seperti mengidentifikasi sentimen, mengelompokkan topik, atau melakukan analisis keterkaitan kata-kata.

    Contoh textual dataset:

    • Teks berita.
    • Ulasan produk.
    • Transkrip wawancara.
    • Tweet di media sosial.
    • Konten web.
    • Dokumen akademik.

    Dalam Tableau, beberapa contoh visualisasi yang dapat dilakukan dengan dataset tekstual meliputi:

    1. Word Cloud: Menganalisis korelasi antara kata-kata dalam teks.
    2. Korelasi Kata-Kata: Melihat bagaimana penggunaan kata-kata tertentu berubah seiring waktu.

Format Dataset

Dalam Tableau, dataset dikelompokkan menjadi dua berdasarkan format dataset yaitu dataset terstruktur dan dataset tidak terstruktur. Berikut adalah beberapa perbedaannya:

Komponen Structured Unstructured
Format Diatur dalam format tabel dengan baris dan kolom Tidak diatur dalam format tabel yang jelas
Keterhubungan dan Relasi Memiliki relasi antara tabel atau kolom Tidak memiliki skema yang terstruktur.
Penyimpanan Membutuhkan lebih sedikit ruang penyimpanan Membutuhkan lebih banyak ruang penyimpanan
Pemrosesan dan Analisis Mendukung analisis data yang lebih mudah. Melibatkan teknik pemrosesan khusus.
Implementasi Data tabular yang memuat data numerik, tanggal, waktu, teks Foto, gambar, suara, video, e-mail, chat

Notes

Dataset terstruktur membutuhkan lebih sedikit ruang penyimpanan dan mendukung analisis yang lebih mudah, sementara dataset tidak terstruktur memerlukan teknik khusus dan lebih banyak ruang penyimpanan.

Visualisasi dengan Tableau

Menghubungkan Data ke Tableau

Untuk memulai visualisasi data, berikut adalah langkah-langkah untuk menghubungkan dataset Superstore yang telah diunduh ke dalam Tableau:

  1. Buka program Tableau yang telah terinstal.
  2. Pada panel "Connect" di sisi kiri halaman utama, pilih tautan "Microsoft Excel" di bawah bagian "To File".
  3. Jendela "Open" akan terbuka, pilih file Excel dataset yang sesuai.
  4. Setelah memilih file, tampilan akan berubah seiring dengan proses impor data. Pilih lembar kerja "Order" dari panel "Sheets" dengan drag-and-drop ke bagian bawah label "Global Superstore".
  5. Setelah memilih lembar kerja "Orders", data dan variabel yang ada dalam lembar kerja "Orders" akan muncul. Untuk memulai proses pembangunan visualisasi data, beralih ke tautan "Sheet 1".

Untuk membuat histogram dari penjualan/Sales:

  1. Klik 2x pada Sheet 1, ganti labelnya menjadi “Histogram of Sales”.
  2. Pilih “Sales” pada panel “Tables” kemudian pada panel “Show Me” pilih ikon . Histogram dari Sales akan muncul.

Membangun Scatter Plot

Untuk membuat scatter plot antara Sales dan Profit:

  1. Pada menu “Worksheet” pilih “New Worksheet” untuk menambahkan sheet baru,

    2. lalu klik 2x pada sheet 2 untuk mengganti label menjadi “Sales vs Profit Scatter Plot”.

  2. Pilih “Profit” dan “Sales” pada panel “Tables” dengan menahan tombol CTRL pada keyboard, pada panel “Show Me” pilih ikon .
  3. Secara default, ukuran (Measure) yang digunakan pada Tableau adalah Sum atau jumlah. Ubah ke dimensi klik dan pilih opsi Dimension pada pop-up menu untuk variabel Sales dan Profit.

Dashboard dalam Tableau

Tableau Dashboard

Dashboard pada Tableau adalah tampilan visual yang terdiri dari berbagai elemen visual seperti grafik, diagram, peta, teks, dan gambar, yang dirancang untuk menyajikan informasi dengan cara yang mudah dipahami dan interaktif. Dashboard Tableau memungkinkan pengguna untuk membuat laporan yang menarik secara visual, menampilkan data yang mudah dicerna, dan membantu memperoleh insight dari data yang kompleks.

Tujuan utama dari dashboard pada Tableau adalah untuk membantu pengambilan keputusan dengan menyajikan data yang kompleks secara intuitif dan mudah dimengerti. Dashboard memungkinkan pengguna untuk berinteraksi dengan data dengan cara mengklik, memfilter, atau memilih elemen visual tertentu untuk melihat informasi yang lebih mendalam atau untuk mendapatkan pandangan yang berbeda terhadap data.

Membangun Dashboard dengan Tableau

Untuk membangun dashboard menggunakan grafik visualisasi yang sudah ada:

  1. Membuat Dashbord Baru:
    1. Pada menu “Dashboard”, pilih “New Dashboard” untuk menambahkan dashboard baru.
    2. Klik dua kali pada “dashboard 1” dan ganti labelnya menjadi “Super Store Dashboard”.
  2. Menyesuaikan Ukuran Dashboard:
    1. Ubah ukuran dashboard menjadi otomatis agar memudahkan dalam mengaturnya sesuai dengan banyaknya visualisasi yang ditampilkan.
    2. Pilih “Size” pada panel “Dashboard”, kemudian pilih “Automatic” pada “Range”.
  3. Menambahkan Visualisasi ke Dashboard:
    1. Untuk membangun dashboard, drag-and-drop visualisasi grafik pada panel “Sheet”.
    2. Sebagai contoh, pada modul ini akan dibangun dashboard dari ketiga visualisasi sebelumnya: histogram, scatter plot, dan box plot.
    3. Pilih salah satu visualisasi yang telah dibangun, drag-and-drop pada panel “drop sheet here”.
    4. Visualisasi pada dashboard akan muncul.
    5. Tambahkan visualisasi lain pada dashboard dengan drag-and-drop seperti cara sebelumnya dan mengatur posisinya sesuai dengan preferensi masing-masing. Misalnya, scatter plot dapat diletakkan di atas histogram.
