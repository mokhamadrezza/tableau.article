Big Data

Pengertian Big Data

Big Data adalah kumpulan data yang sangat besar atau kompleks, yang sulit diolah dengan metode tradisional. Jumlah data yang dihasilkan oleh berbagai sumber, mulai dari transaksi bisnis hingga interaksi online, terus meningkat seiring dengan kemajuan teknologi. Konsep Big Data melibatkan tiga tahap utama: pemrosesan, analisis, dan visualisasi data dalam skala besar.

Pemrosesan dan Data Analisis Big Data

Teknologi modern memungkinkan pengumpulan dan penyimpanan data dalam jumlah besar. Pemrosesan data yang terstruktur dan tidak terstruktur memungkinkan penggalian wawasan, identifikasi pola, dan pengungkapan tren yang tidak terlihat pada pandangan pertama. Big Data juga memungkinkan analisis yang lebih mendalam untuk pengambilan keputusan yang lebih baik dan inovatif.

Perbedaan Big Data dan Small Data

Perbedaan utama antara Big Data dan Small Data adalah dalam hal volume, kecepatan, variasi, dan kualitas data:

Komponen Big Data Small Data
Volume Sangat besar, mencapai petabyte atau eksabyte Lebih terkendali
Velocity Kecepatan pertumbuhan data sangat cepat Pertumbuhan data lebih lambat dan dapat diprediksi
Variety Beragam dalam jenis maupun format Cencerung lebih homogen
Veracity Tantangan dalam kualitas dan keandalan data Kualitas data lebih tinggi

Mengelola Big Data dalam Tableau

Teknik Mengelola Big Data di Tableau

  1. Tableau Extract: Teknik ini mengoptimalkan kinerja dengan mengambil bagian atau seluruh data dari sumbernya dan menyimpannya dalam format file Tableau extract (.tde atau .hyper). Teknik ini memudahkan pengolahan data besar dan kompleks dengan cara membuat salinan lokal dari data yang dibutuhkan.
  2. Data Blending: Teknik ini memungkinkan penggabungan data dari berbagai sumber untuk membuat visualisasi data yang lebih kaya dan informatif. Tableau menggabungkan data dari sumber yang berbeda berdasarkan hubungan yang telah ditentukan.

Data Extracts

Pengertian Tableau Data Extract (TDE)

TDE adalah format penyimpanan data yang dioptimalkan dan dikompresi khusus untuk aplikasi Tableau. TDE dapat dihasilkan dari berbagai sumber data seperti file CSV, Excel, dan database relasional. Ekstrak data ini memungkinkan pengguna untuk menganalisis data dengan cepat dan efisien.

Keuntungan Tableau Data Extracts:

  1. Kinerja yang Lebih Cepat: Visualisasi merespons lebih cepat karena data telah diolah dan dioptimalkan.
  2. Pemrosesan Lebih Efisien: Sebagian besar operasi pemrosesan data telah selesai sebelum visualisasi dilakukan, mengurangi beban pemrosesan real-time.
  3. Akses Offline: Pengguna dapat bekerja dengan ekstrak tanpa terhubung langsung ke sumber data.
  4. Optimasi Kolom: Pengguna dapat mengoptimalkan kolom untuk jenis data tertentu, mengurangi kebutuhan penyimpanan dan meningkatkan kinerja analisis.
  5. Pembaruan Data yang Fleksibel: Pengguna dapat memperbarui ekstrak sesuai jadwal yang ditentukan, memilih antara pembaruan penuh atau inkremental.

Implementasi Tableau Data Extract

  1. Menghubungkan ke Data: Setelah terhubung ke data, pilih extract di sudut kanan atas dan klik Edit.
  2. Konfigurasi extract: Tentukan bagaimana data extract harus disimpan, seberapa banyak data yang akan diekstrak, agregasi data, dan sampling data.
  3. Membuat extract: Klik OK, lalu pilih sheet tab untuk memulai pembuatan extract

Data Blending

Pengertian Tableau Data Blending

Tableau Data Blending adalah teknik yang digunakan untuk menggabungkan data dari sumber yang berbeda dalam satu analisis visual. Teknik ini memungkinkan pengguna menciptakan visualisasi yang lebih kaya dan mendalam dengan mengintegrasikan informasi dari beberapa sumber data yang mungkin memiliki struktur atau tipe yang berbeda.

Notes

Menjawab Pertanyaan seperti “Apa yang ingin ditunjukkan?” dapat membantu memilih jenis diagram yang tepat.

Keuntungan Tableau Data Blending

  1. Integrasi Sumber Data Berbeda: Menggabungkan data dari berbagai sumber dengan struktur dan format yang berbeda.
  2. Wawasan Lebih Komprehensif: Mengidentifikasi hubungan dan tren yang tidak terlihat jika menganalisis masing-masing sumber secara terpisah.
  3. Memperkaya Visualisasi: Membuat presentasi dan laporan yang lebih menarik dan informatif.
  4. Mengatasi Keterbatasan Sumber Tunggal: Menambahkan sumber tambahan yang relevan untuk pemahaman yang lebih baik.
  5. Analisis Lintas Dimensi: Menganalisis data dari berbagai sudut pandang dan memahami dampak interaksi antara variabel-variabel.

Implementasi Tableau Data Blending

  1. Menghubungkan Workbook ke Sumber Data: Pastikan workbook memiliki beberapa sumber data, misalnya "Target Sales" sebagai Primary Data.
  2. Menambahkan Sumber Data Kedua: Tambahkan "Sales Profit" sebagai Secondary Data, dengan cara pergi ke Data → Sumber Data Baru → Microsoft Excel. Pada modul ini akan digunakan data “Sales Profit”.
  3. Ubah Tipe Data: Ubah tipe data data “Date” pada “Sales Profit” dan “Target Sales” yang semula string menjadi date, dengan cara klik kanan "Date" → Change Data Type → Date.
  4. Membuat Line Chart: Pilih data source “Sales Profit”, pilih "Date" dan "Sales", kemudian buat single line chart.
  5. Ubah Periode Waktu: Karena periode waktu default di Tableau adalah Tahunan, data hanya muncul sebagai titik. Untuk mengubahnya menjadi bulanan, klik "Year (date)" di panel "Columns" dan pilih "Month".
  6. Menambahkan Data “Target Sales”: Pilih “Target Sales” dan drag-and-drop “Target Sales” ke panel "Rows". Line chart baru akan muncul secara otomatis di bawah.

    7. Dengan memahami dan mengimplementasikan teknik-teknik ini, pengguna dapat memanfaatkan fitur-fitur Tableau untuk membuat visualisasi data yang detail, interaktif, dan memberikan wawasan yang mendalam dari Big Data.
